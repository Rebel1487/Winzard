## How to make them look professional
- Capture **only the WPI window** (not the whole desktop): focus the window and press **Alt + PrtScn**, or use **Win + Shift + S** and select the window.
- Save as **PNG** (sharper than JPG for UIs).
- Use the **same window size** in all of them so the mosaic looks uniform.
- The **Blue (Chris Titus)** and **Dark** themes look great on GitHub.
- Take them **now**, with the fixed version (correct tooltips/separators, status counters, guided ISO wizard, clean repair console).
